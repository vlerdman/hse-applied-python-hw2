# Telegram Fitness Bot HW2 

## Commands list

- /start - start conversation

- /set_profile - set up profile

- /log_water <ml> - log water consumption

- /log_food <food> - log food consumption

- /log_workout <type> - log workout

- /check_progress - check progress

- /help - commands list
